# Cloud Manager 2022.12.0 릴리스 노트 {#release-notes}

이 페이지는 [!UICONTROL Cloud Manager] 2022.12.0에 대한 릴리스 정보를 설명합니다.

>[!NOTE]
>
>AEM as a Cloud Service의 Cloud Manager에 대한 최신 릴리스 정보는 AEM as a Cloud Service의 현재 릴리스 정보의 [Cloud Manager](https://experienceleague.adobe.com/docs/experience-manager-cloud-service/content/implementing/using-cloud-manager/release-notes-cloud-manager/release-notes-cm-current.html)을(를) 참조하십시오.

## 릴리스 날짜 {#release-date}

[!UICONTROL Cloud Manager] 2022.12.0의 릴리스 날짜는 2022년 11월 29일입니다. 다음 릴리스는 2023년 1월 19일 금요일에 예정되어 있습니다.

## 새로운 기능 {#what-is-new}

* 이제 다중 지역 자동 확장 기능을 사용할 수 있습니다. 특정 제한 사항이 적용될 수 있습니다.
   * 이 변경 사항은 2022.12.0 릴리스 이후 4~6주에 걸쳐 단계적으로 출시될 예정입니다.
* [Cloud Manager 파이프라인](/help/overview/ci-cd-pipelines.md)의 가용성 및 안정성이 개선되었습니다.
